Key Functionalities

Smart Borrowing System

The core of any library. This feature allows librarians to check out items to specific patrons. It records the borrower's name and the transaction date, instantly updating the item's status to "Checked Out" in the database.

Live Statistics

Data at a glance. The analytics module queries the backend to generate a real-time summary of the library's inventory, breaking down totals by category (Books, Films, Magazines) and tracking active loans.

Inventory Control

A dedicated interface for expanding the collection. This form ensures data integrity by validating required fields like Author and Publication Date before committing the new entry to the permanent JSON storage.

Backend Logic (Flask)

A custom-styled desktop interface built with Tkinter and TTK. Acts as the client, sending HTTP requests to the Flask server to retrieve data.
