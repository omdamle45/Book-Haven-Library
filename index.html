Book Haven Library Manager

Project Overview

Book Haven is a desktop-based library management system that helps manage borrowing, inventory, and statistics. It is designed to give students and librarians an easy-to-use interface with backend support using Python Flask.

Key Functionalities

Smart Borrowing System

Allows librarians to check out media, store borrower info, and update the item status instantly.

Category Filtering

Quickly narrow the library list view by category (Books, Films, Magazines). This feature utilizes a dedicated API endpoint to retrieve specific subsets of data.

Edit Existing Media

Easily modify details of any existing book, film, or magazine. Update titles, authors, publication dates, and categories with simple form inputs.

Live Statistics

Displays total items, categories, and currently borrowed media using backend calculations.

Inventory Control

Add new media with validation checks for author, date, and category before committing to storage.

Installation Guide

  1. Install Python (3.10 or above).
  2. Install required libraries: 
    pip install flask requests
  3. Start the Flask backend: 
    python backend.py
  4. Run the Tkinter frontend: 
    python frontend.py

API Documentation

GET /media

Returns all media items.

GET /media/category/<cat>

Returns items filtered by Category (e.g., Book, Film, Magazine).

POST /media

Add a new item to the library.

PUT /media/<id>

Update details of an existing media item.

POST /media/<id>/borrow

Mark an item as borrowed and store borrower information.

POST /media/<id>/return

Change the status back to available.

GET /stats

Return total items, categories, and currently borrowed count.
