Project Overview

Smart Library Management

This application serves as a complete solution for managing media in a library. It moves beyond simple storage by offering features like borrowing tracking, automatic data persistence, and a modern graphical interface.

  • Full CRUD: Create, Read, Update, and Delete items.
  • Smart Search: Find books even with partial names.
  • Borrow System: Track who borrowed a book and when.
  • Analytics: View statistics on library holdings.
