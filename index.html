Book Haven Library Manager

A desktop library management application created by Om Damle, Mechatronics Engineering student at SRH Berlin.

View Source Code

Key Functionalities

Smart Borrowing System

The app allows borrowers to check out media and updates the backend instantly.

Live Statistics

Shows item counts, borrowed items and categories through backend calculations.

Inventory Control

Add new media with validation for authors, dates and categories.

Frontend (Tkinter)

The frontend is created using Python Tkinter. It provides a simple and clean interface for adding items, borrowing books and viewing data.

Backend (Flask)

The backend is built using Flask. It provides API routes for adding items, updating status and showing statistics.

Installation Guide

  1. Install Python 3.10 or newer.
  2. Install dependencies: 
    pip install flask requests
  3. Start the Flask backend: 
    python backend.py
  4. Open a second terminal and start the Tkinter frontend: 
    python frontend.py

API Documentation

GET /items

Returns all media items.

POST /add

Add a new media item.

PUT /borrow/<id>

Mark an item as borrowed.

PUT /return/<id>

Mark the item as returned.

GET /stats

Shows library statistics.
